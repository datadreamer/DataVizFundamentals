Evaluating Data


  1. Dimensionality.

    A measure of the number of variable properties of a given object or data sample.

  2. Range.

    The measure between the minimum and maximum values for a given variable property.

  3. Granularity.

    The resolution of a set of data samples or the possible values within a variable property.

  1. Correlation and Causation.

    The phenomena of variable properties fluxuating in sync; by relationship or effect.

  2. Regression Analysis.

    The statistical process for estimating the relationships among variable properties.
